# [traveller](https://github.com/xiangwenhu/traveler)

## 描述
时光荏苒人易老，往事如烟梦已遐。 莫叹浮生多苦短，且将旅行趁年华。  
记录每次旅行，通过图片，视频等记录足迹，提供演示能力，通过报表知道旅行概况，接入在线剪辑能力，快速生成精彩集锦。

  - 用户管理
  - 标签管理
  - 5A管理
  - 高校管理
- 旅行

  - 旅行计划

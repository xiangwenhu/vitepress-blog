# yapi 转为 petal-service

## 描述
* 起因：公司的部分服务文档都是通过yapi发布的，前端服务请求，需要手写类型typescript和请求逻辑，麻烦。   
yapi转为typescript编写的的服务代码。升级愿景转为基于[petal-service](https://github.com/xiangwenhu/petal-service)框架的服务请求。
